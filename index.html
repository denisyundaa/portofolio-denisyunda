About Me

Hello, I'm Denis Yunda.

As an Informatics Engineering student currently in my 6th semester at Universitas Pamulang, I believe in the power of visuals and technology to tell compelling stories. This belief fuels my dedication as a Front-End Developer, where I focus on designing and building websites that are not only functional but also visually delightful. I've already gained some experience in creating simple websites and am passionate about continuously developing my skills in this dynamic field.

Beyond the world of code, the camera lens is my close companion. I actively work as a freelance Videographer, Camera Operator, and Video Editor. In these roles, I find joy in framing the world, capturing emotions, and transforming them into impactful visual narratives. Whether I'm developing an elegant user interface or producing cinematic video, my goal is always the same: to create memorable, high-quality work.

Movie & TV Show List

Category: Web Design

Movie & TV Show recommendation web app using TheMovieDB API. Built with HTML, CSS, Bootstrap, and JavaScript.
This project is a frontend web application that provides popular movie and TV show recommendations and allows users to search for specific titles. This project is built using HTML, CSS (with Bootstrap 5), and vanilla JavaScript, and dynamically fetches data from TheMovieDB (TMDB) API.

  • Dual Categories: Choose to explore "Movies" or "TV Shows."
  • Currently Popular: Displays a list of the most popular movies or TV shows by default.
  • Dynamic Search: Search for movies or TV shows by title.
